1. 문장들을 묶어놓은 것(작업단위)

2. 값(입력)을 받아서 처리하고, 결과를 반환(출력)

   ```java
   int add(int x, int y) 	// 선언부
   {						// 구현부
       int result = x + y;
       
       return result; // 결과물 반환
   }
   ```

- 장점
  - 코드의 중복을 줄임
  - 관리가 쉽다
  - 재사용
  - 간결해서 이해하기 쉽다

- 지역변수(lv): 메서드 내에 선언된 변수
