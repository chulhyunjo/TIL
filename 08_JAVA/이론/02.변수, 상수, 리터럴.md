# 02.변수, 상수, 리터럴

- 변수: 하나의 값을 저장하기 위한 공간
- 상수: 한 번만 값을 저장 가능한 변수(final)
- 리터럴: 그 자체로 값을 의미하는 것 = 기존의 상수

```java
int score = 100;
score = 200;

final int MAX = 100; // MAX는 상수
MAX = 200; // 에러

char ch = 'A';
String str = 'abc';
```
