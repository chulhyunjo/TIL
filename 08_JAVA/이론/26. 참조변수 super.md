# 26. 참조변수 super

- 객체 자신을 가리키는 참조변수, 인스턴스 메서드(생성자)내에만 존재(static 사용 불가)
- 조상의 멤버를 자신의 멤버와 구별할 때 사용



```java
class Ex{
    public static void main(String args[]){
        Child c = new Child();
        c.method();
    }
}

class Parent {
    int x = 10; // super.x
}

class Child extends Parent {
    int x = 20;
    
    void method() {
        sout("x=" + x);				// 20
        sout("this.x="+ this.x);	// 20
        sout("super.x="+ super.x);	// 10
    }
}
```
