# Jupyter 단축키

|       단축키       |               기능               |
| :----------------: | :------------------------------: |
|         H          |         단축키 목록 출력         |
|         A          |           상단 셀 생성           |
|         B          |           하단 셀 생성           |
|   클릭  or enter   |         Edit Mode로 전환         |
|        esc         |       Command Mode로 전환        |
| 가위 아이콘 or :x: |             셀 삭제              |
|    ctrl + enter    |           현재 셀 실행           |
|   shift + enter    | 실행 + 다음 셀 선택(없다면 생성) |
|    alt + enter     |       실행 + 다음 셀 생성        |
|         M          |        마크다운 셀로 변경        |
|         Y          |          코드 셀로 변경          |
|   ctrl + s or s    |               저장               |
